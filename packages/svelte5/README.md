# @kksh/svelte5

## Installation

```bash
pnpm add @kksh/svelte5
```

### Path Alias Configuration

Also configure `tailwind.config.ts`

```ts
content: [
    ...
    "./node_modules/@kksh/svelte5/src/**/*.{html,js,svelte,ts}"
],
```

### Peer Dependencies

Make sure the following packages are also installed

- lucide-svelte
- svelte-sonner
